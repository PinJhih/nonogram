# Nonogram Solver

## Build

```bash
make all
```

## Run

預設會讀取工作目錄下的 *input.txt* 作為輸入
```
cd ./bin
./nonogram
```

或是在 arguments 中指定要開啟的檔案
```
cd ./bin
./nonogram <your_input_file>
```
